

## ğŸš€ Deploying to GitHub Pages

1. Install `gh-pages`:

```bash
npm install gh-pages --save-dev
```

2. Add this to `package.json`:

```json
"homepage": "https://nach1ket03.github.io/edutech",
"scripts": {
  "predeploy": "npm run build",
  "deploy": "gh-pages -d dist"
}
```

3. Deploy with:

```bash
npm run deploy
```
